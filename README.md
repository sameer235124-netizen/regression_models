# Logistic Regression Model (R)

## 📌 Overview
This project implements a logistic regression model in R to predict the likelihood of a heart attack based on patient data.

## 📂 Files
- logistic_regression.R → main script containing model code
- heart_attack.csv → dataset used for training and testing

## ⚙️ Requirements
- R
- Packages:
  - caret
  - pROC
  - caTools

Install packages if needed:
```r
install.packages(c("caret", "pROC", "caTools"))
